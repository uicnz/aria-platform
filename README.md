## Aria Platform – Support & Discussion

Welcome! This is a support space for all things **Aria**.

- **[Start a discussion](../../discussions)** – share ideas, ask questions, or connect with others
- **[Report a problem](../../issues/new/choose)** – something not working as expected? Let us know
- **[Request a feature](../../issues/new/choose)** – tell us what would make Aria better for you
- **[Browse the wiki](../../wiki)** – guides, FAQs, and helpful information

---

If something is weird, difficult, or broken, please share.

When you open an issue, there's a few basic questions to understand. There's no wrong bad questions.

---

## How to get help

| I want to… | Where to go |
|---|---|
| Ask a question or start a conversation | [Discussions](../../discussions) |
| Report a bug or problem | [New Issue → Report a Problem](../../issues/new?template=bug_report.yml) |
| Request a new feature or improvement | [New Issue → Feature Request](../../issues/new?template=feature_request.yml) |
| Share general feedback | [New Issue → General Feedback](../../issues/new?template=feedback.yml) |
| Read guides and FAQs | [Wiki](../../wiki) |

---

*This repository is the public-facing support channel for the private [uicnz/aria](https://github.com/uicnz/aria) codebase.*
